# GA 360

Portal corporativo para gestão estratégica, reuniões, governança e performance.

## Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Execução local

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Deploy

O projeto está preparado para deploy em ambientes estáticos/SSR compatíveis com Vite (ex.: Vercel).
